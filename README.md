## РАЗРАБОТКА

запуск dev сервера:
`npm run dev`

Для запуска разработки необходимо создать файл `.env` и добавить в него переменные

```javascript
TELEGRAM_TOKEN={{YOUR_TELEGRAM_TOKEN}}
DB_URL={{MONGO_DB_URL}}
```
